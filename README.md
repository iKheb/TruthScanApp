# TruthScanApp

Analiza conversaciones, relaciones y señales ocultas con IA para obtener claridad emocional rápida y compartible.

## Ejecutar

```bash
npm install
npm run dev
```

## Configuración

1. Copia `.env.example` a `.env.local`.
2. Completa tus variables de Firebase y proveedor IA.
3. No subas `.env.local` al repositorio.
